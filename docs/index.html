TinyCMS 1.4 Documentation

Thanks for downloading TinyCMS 1.4, we hope you enjoy it.

Installing TinyCMS

To install TinyCMS, upload the files to your server and you're ready to roll. Visit YourSite.com/admin/ and use the default login details. The default username is admin and the default password is pass.

Installing themes/templates

At this time, we're still creating a great guide for themes. Sadly, we had to push the 1.4 update so we didn't get around to improving the templates. Check out our themes page for some starters.

Changes to file locations

Note for existing themes. In 1.3, we moved the JS files from yoursite.com/js to yoursite.com/admin/js to clean up the directories. If you're using a theme from our website, ensure the directories are correct or the site may not work correctly.

Notes

TinyCMS 1.4 includes a new feature, drafts! Save your pages without publishing them so you don't have to worry about them going live before they're ready.
